MPC Path-tracking & Obstacle Avoidance for Mobile Robot

Python Control Systems Model Predictive Control Robotics Simulation Dynamics Linearisation Trajectory Planning Optimization

Project Description

This project involved building a custom Model Predictive Control (MPC) framework for a two-wheeled differential drive robot. The controller was designed to track a predefined path while avoiding obstacles, accounting for system dynamics and real-world constraints. I developed both nonlinear and linearized control strategies and validated them through Python-based simulations.

Key Responsibilities

  • Developed a hybrid control strategy by generating a reference trajectory offline using nonlinear MPC, then transitioning to a linearized MPC model during runtime to enhance computational efficiency.
  • Applied Zero-Order Hold for linear system discretization and Forward Euler for nonlinear system discretization.
  • Simulated control strategies in Python, validating performance under noise, model mismatch, and nonlinear dynamics.
